# Proyecto de Fresh-prince

Este proyecto se lograron las siguientes funcionalidades:

1. **Inicio de sesión**
2. **Landing Page**
3. **Ver catálogo**
4. **Agregar y eliminar productos del carrito**

## Funcionalidades Desarrolladas
### 1. Inicio de Sesión
Se ha implementado un sistema de autenticación de usuarios que permite a los usuarios iniciar sesión con sus credenciales.

### 2. Landing Page
Incluye información los productos.

### 3. Ver Catálogo
Los usuarios pueden ver el catálogo completo de productos disponibles.

### 4. Agregar y Eliminar Productos del Carrito
Se ha implementado una funcionalidad de agregar al carrito que permite a los usuarios agregar productos al carrito y eliminarlos.

## Instalación
Para instalar y ejecutar el proyecto localmente, sigue estos pasos:
1. Clona el repositorio:[repositorio](https://github.com/JCampo1502/Fresh-prince.git)
